## Boundary

The D5J runner is a guarded one-job Firestore shadow smoke tool. It reuses the D5F-D5I schema and validator, the D2 job-store contract, D3 report-only reconciliation language, and the D5E durable shadow state shape. It does not read Gmail, Drive, Sheets, or Apps Script data.
