D5D-R keeps the temporary read-only Apps Script function but simplifies owner setup to exactly one Script Property: `SGDS_D5D_GMAIL_THREAD_ID`. The previous XML/PDF file IDs, expected line count, expected hashes, invoice identity hash, invoice key hash, and commit plan hash are no longer owner inputs.


The executor derives invoice identity from seller tax code, symbol, invoice number, and issue date. It derives legacy `InvoiceKey` with the runtime invoice-key helper and derives line `HashIndex` values with the runtime invoice item hash helper. Hoa-Don supplies XML/PDF Drive file IDs after lookup by derived invoice key.
