## Purpose

D5F defines what a future production Firestore shadow write may persist. Firestore is a durable control plane only. It does not replace the Google Sheets canonical ledger, Google Drive XML/PDF evidence store, or Gmail source.

```text
SHADOW_WRITE_SCOPE=invoiceJobs;events;reconciliationReports;safe source projections
GOOGLE_SHEETS_CANONICAL_LEDGER=PRESERVED
GOOGLE_DRIVE_EVIDENCE_STORE=PRESERVED
GMAIL_SOURCE=PRESERVED
```

## Write Contract

Future production shadow write may create or reuse one deterministic `invoiceJobs/{jobId}` document, save one immutable commit plan, append bounded audit events, and save report-only reconciliation reports.

```text
IDEMPOTENCY_CONTRACT=DETERMINISTIC_JOB_ID_AND_IDEMPOTENCY_KEYS
VERSIONING_CONTRACT=OPTIMISTIC_VERSION_PLUS_IMMUTABLE_COMMIT_PLAN
IMMUTABLE_COMMIT_PLAN_RULE=YES
APPEND_ONLY_EVENTS=YES
REPORT_ONLY_RECONCILIATION=YES
EXECUTION_MODE=SHADOW
PRODUCTION_MUTATION_ALLOWED=false
AUTOMATIC_REPAIR=DISABLED
BATCH_ACTIVATION=NOT_APPROVED
BACKFILL=NOT_APPROVED
```
